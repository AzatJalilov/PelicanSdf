METHOD

Benchmark methodology

The host supplies a blank canvas and no scene helpers. The submitted module owns geometry, shaders, camera state, rendering, and interaction. The canonical prompt explicitly states that frame rate will be measured.

  1. 01
    Generate

    Fresh conversation, byte-exact prompt, first response only, no repair pass.

  2. 02
    Validate

    Compile and render checks, twelve camera views, close zoom, and required input paths.

  3. 03
    Measure

    GPU timer queries where available; synchronized wall-clock timing otherwise.
